# Running the "get" code Locally
Requires VSCode and Python 3.x.x

## Virtual Environment
Create a Virtual Environment
```
python3 -m venv venv
```

Set up Virtual Environment (Windows)
```./venv/Scripts/activate```
Set up Virtual Environment (Linux / Mac)

## Install Dependencies
```
pip install --upgarde pip
pip install notebook jupyterlab requests ipynb ipython pandas py3dmodel biopython
```

## Create a Jupyter Notebook
Open the Command Pallete (Ctrl + Shift + P), and search for `Jupyter: Create New Blank Notebook`.

Select local Kernel Source (should include ./venv/ somewhere)

## Run
In ```main.ipynb```, select "Run All", and scroll down for the output
